Mini Jeu

Speed Letter

Speed Letter est un mini jeu que j'ai réalisé entièrement avec le langage JavaScript, le style également.
Le premier écran contient le titre les règles du jeu et un boutton pour commencer à jouer.

Le but du jeu est simple, une lettre aléatoire apparaît à l'ecran et le joueur doit appuyer sur la touche qui convient, il a 3 secondes pour appuyer sur la bonne touche, si le temps s'écroule il a perdu.

Si le joueur perd le troisème écran apparaît avec le message Game Over ainsi que le nombre de point qu'il a remporter.

Il reste quelques beugs à corriger.

Technologies utilisées
